Reinforcement Learning for NLP (RL-NLP) is a package that facilitates the use of reinforcement learning for NLP. 

## Upcoming Features
- APIs for defining RL's state space, step and actions
- RL-base dialouge manager

## Usage

Install the package:

`pip3 install rl-nlp`
